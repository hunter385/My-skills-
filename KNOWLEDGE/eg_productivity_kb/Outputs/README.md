# Outputs — productivity_kb

This folder holds generated answers, reports, charts, and other deliverables that the librarian produces against this KB.

When you ask the librarian a question, it writes a report here following the Question Report Protocol in the KB's `CLAUDE.md`. Reports stay here as the working archive. The librarian may propose promoting a report into the Wiki when its synthesis is foundational enough to belong in the long-term knowledge layer.

Files use the naming convention `YYYY-MM-DD_query-slug.md` (kebab-case slug, lowercase).

This README ships with the template so the folder exists as part of the system. Once your first output lands here, you can leave this file or delete it — either is fine.
