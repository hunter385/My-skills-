# **Output Guidelines**
### Format
- Use Markdown headings for each section.
- Bold key phrases for clarity.
- Write in short paragraphs and bulleted lists.
- Include timestamps, B-roll ideas, and on-screen text cues.
### Style & Tone
- Conversational, clear, motivating.
- Professional but warm.
- Avoid em dashes.
- Aim for clarity over cleverness.
