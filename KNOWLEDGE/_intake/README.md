# _intake — staging folder for second-brain imports

Drop Obsidian markdown files (and any folders / attachments) here. Structure doesn't matter — keep your existing folder layout, flatten it, dump everything in. Whatever.

I'll triage from here into the right KBs:

- RSG frameworks + teaching content → `KNOWLEDGE/rsg_kb/RAW/`
- Book notes + reading capture → `KNOWLEDGE/reading_kb/RAW/`
- Personal LifePlan + family rhythms → `KNOWLEDGE/lifeplan_kb/RAW/`
- Half-formed ideas → reviewed; some become framework candidates, some go to `WORK AREAS/`, some stay as raw capture.

## How to use

1. Drop files here (drag from Obsidian, or copy the vault folder).
2. Tell me: "Triage my intake."
3. I'll scan, propose a destination for each file (or folder of files), and confirm before moving anything.
4. After triage, this folder gets emptied automatically — files are never deleted, just moved into their KB home.

## What I won't do

- I won't delete anything.
- I won't move anything without showing you the plan first.
- I won't ingest anything until you confirm the triage map.
