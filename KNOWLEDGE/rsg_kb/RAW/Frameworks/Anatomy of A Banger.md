## Core Idea
| HOOK                                     | BUILD                              | PAYOFF                                 | INVITATION                                        |
| ---------------------------------------- | ---------------------------------- | -------------------------------------- | ------------------------------------------------- |
| what is the thing that stops the scroll? | tell a story                       | the point                              | an ask to "do what comes next"                    |
| that gets them to pay attention?         | give them **steps**                | the insight                            | call to engagement (comment, like, share, follow) |
| capture attention                        | give them a **list**               | the punchline to the joke              | call to action (engage in your sales process)     |
|                                          |                                    | the “ah ha”                            |                                                   |
|                                          | a story                            | a new way **to do** something          |                                                   |
|                                          | a list (3 things you need to know) | a new way **to think** about something |                                                   |
|                                          | steps (3 steps / 5 steps)          |                                        |                                                   |

The Hook is probably calling out the <mark style="background: #FF5582A6;">old way</mark>
