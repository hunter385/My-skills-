Getting set up

Add the Better Creating Marketplace

CoWork OS plugins install from the Better Creating marketplace. The marketplace will always have the latest plugins available for your system — here's how to set it up.

  • Step 1. Open Customise (sliders icon) > Plugins > click + next to "Personal plugins" > Add marketplace
  • Step 2. Type bettercreating/cowork-os-plugins in the URL field > click Sync
  • Step 3. Click Browse plugins > go to Personal tab > click + on each plugin you want

About the safety warning: Anthropic shows this for every third-party plugin — it's not specific to CoWork OS. These plugins were built by Simon at Better Creating and only contain text-based instructions and connections to your own tools. Nothing risky.

Using Claude Code?

Same Process, Different Tab

If you're using CoWork OS in the Claude Code tab instead of CoWork, the process is the same — open Customise > Plugins, add the Better Creating marketplace, and install your plugins from there.

Troubleshooting

Plugins don't appear?

Double-check the marketplace URL is exactly bettercreating/cowork-os-plugins. If they installed but skills don't show, close and reopen the Claude app. Still stuck? Email notion@bettercreating.com.
