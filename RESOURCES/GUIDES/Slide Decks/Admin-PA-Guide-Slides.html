The foundation

What Is Admin PA?

Admin PA is two things working together. First, a dedicated work area inside CoWork OS — folders where general tasks, system reviews, and day-to-day admin live. Second, an optional Personal Assistant plugin — a dedicated instruction set that turns this area into a conversational personal assistant. The folders work on their own from day one. The plugin activates when you're ready.

Admin PA lives at WORK AREAS/Admin-PA/. When you ask Claude for something that doesn't clearly belong elsewhere, it goes here.

Right now

What It Does Right Now

Quick Tasks

One-offs that don't fit in a project. Change a password, find a file, draft an email.

System Reviews

Your folder structure, automation health, what's working and what's not.

First Week Guide Outputs

Your onboarding documents, setup checklists, and reference materials.

General Admin

Files, notes, and catch-all documents that don't belong in a specific project.

Folders are automatically named using kebab-case. The system adapts as you work.

Optional activation

The Personal Assistant Plugin

The optional instruction set. Once installed, you narrate your day and it organises. The core idea: your assistant knows what matters — your projects, priorities, people — and surfaces what you need when you need it.

Example: Sarah starts a meeting recap as "Had call with acme about roadmap." The plugin pulls the Acme project, checks recent updates, flags action items, and files the note automatically.

Captain's Log

Daily narration in your own words. The system structures it.

Task Tracking

Extract and organise what you need to do from loose notes.

Contact CRM

People, their context, and your history. Always searchable.
