# Personal Assistant Plugin

**Version:** 1.0.0
**Author:** Better Creating

Turns your Admin-PA work area into an active personal assistant. Just talk to it — it tracks tasks, logs your day, remembers contacts, captures preferences, and briefs you each morning.

## What's Inside

- **Setup wizard** — interactive setup that creates your PA file structure, scheduled tasks, and CLAUDE.md additions
- **Daily briefing** — morning briefing and end-of-day summary with optional reflection
- **5 slash commands** — /log, /tasks, /briefing, /eod, /tidy
- **Reference files** — format specs, task extraction rules, contact structure, system review hooks

## Getting Started

1. Drag this .plugin file into any CoWork session
2. The setup wizard runs automatically
3. Start talking about your day

## Requirements

- CoWork OS v8 or later
- Admin-PA work area (included in CoWork OS by default)

## Optional Integrations

- **Google Calendar** — enables calendar events in your daily briefing
- **Gmail** — enables email summaries in your briefing (future)
